## הרצה מקומית

1. לשכפל את הפרויקט או להוריד כ־ZIP.
2. לפתוח את `index.html` בדפדפן.

או עם שרת מקומי (לדוגמה):

```bash
# עם Python 3
python -m http.server 8000

# עם npx
npx serve
```

ואז לפתוח: `http://localhost:8000`
